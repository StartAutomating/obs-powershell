Get-OBSStudioModeEnabled
------------------------
### Synopsis
Get-OBSStudioModeEnabled : GetStudioModeEnabled

---
### Description

Gets whether studio is enabled.


Get-OBSStudioModeEnabled calls the OBS WebSocket with a request of type GetStudioModeEnabled.
