Get-OBSHotkey
-------------
### Synopsis
Get-OBSHotkey : GetHotkeyList

---
### Description

Gets an array of all hotkey names in OBS


Get-OBSHotkey calls the OBS WebSocket with a request of type GetHotkeyList.
