Stop-OBSReplayBuffer
--------------------
### Synopsis
Stop-OBSReplayBuffer : StopReplayBuffer

---
### Description

Stops the replay buffer output.


Stop-OBSReplayBuffer calls the OBS WebSocket with a request of type StopReplayBuffer.
