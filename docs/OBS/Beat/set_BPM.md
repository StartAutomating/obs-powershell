set_BPM
-------

### Synopsis
Sets the BPM

---

### Description

Set the Beats Per Minute (BPM) of obs-powershell.

This can be used to time effects to a beat.

---

### Parameters
#### **BPM**
The new BPM

|Type      |Required|Position|PipelineInput|
|----------|--------|--------|-------------|
|`[Double]`|false   |1       |false        |
