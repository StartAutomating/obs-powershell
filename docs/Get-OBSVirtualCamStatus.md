Get-OBSVirtualCamStatus
-----------------------
### Synopsis
Get-OBSVirtualCamStatus : GetVirtualCamStatus

---
### Description

Gets the status of the virtualcam output.


Get-OBSVirtualCamStatus calls the OBS WebSocket with a request of type GetVirtualCamStatus.
