OBS.PowerShell.Effect.Step()
----------------------------

### Synopsis
Steps thru an effect

---

### Description

Steps thru an effect.

This will send individual messages from an effect, without sleeps.

---

### Parameters
#### **StepCount**
The step count

|Type     |Required|Position|PipelineInput|Aliases|
|---------|--------|--------|-------------|-------|
|`[Int32]`|false   |1       |false        |Ticks  |
