Switch-OBSRecordPause
---------------------
### Synopsis
Switch-OBSRecordPause : ToggleRecordPause

---
### Description

Toggles pause on the record output.


Switch-OBSRecordPause calls the OBS WebSocket with a request of type ToggleRecordPause.
