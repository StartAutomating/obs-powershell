Start-OBSReplayBuffer
---------------------
### Synopsis
Start-OBSReplayBuffer : StartReplayBuffer

---
### Description

Starts the replay buffer output.


Start-OBSReplayBuffer calls the OBS WebSocket with a request of type StartReplayBuffer.
