Get-OBSSpecialInputs
--------------------
### Synopsis
Get-OBSSpecialInputs : GetSpecialInputs

---
### Description

Gets the names of all special inputs.


Get-OBSSpecialInputs calls the OBS WebSocket with a request of type GetSpecialInputs.
