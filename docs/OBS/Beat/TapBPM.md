OBS.Beat.TapBPM()
-----------------

### Synopsis
Tap BPM

---

### Description

Tap out a BPM by pressing ENTER on N beats.

The BPM will be set to the average time between taps, and the beat will be started.

---

### Parameters
#### **TapCount**
The number of taps.

|Type     |Required|Position|PipelineInput|
|---------|--------|--------|-------------|
|`[Int32]`|false   |1       |false        |
