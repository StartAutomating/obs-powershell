set_Timer
---------

### Synopsis
Sets the Beat Timer

---

### Description

Sets the Beat Timer.

If no timer exists, one is created.  Otherwise, the interval is updated.

---

### Parameters
#### **Interval**
The new timer interval.

|Type        |Required|Position|PipelineInput|
|------------|--------|--------|-------------|
|`[TimeSpan]`|false   |1       |false        |

---

### Outputs
* [TimeSpan]
