Switch-OBSVirtualCam
--------------------
### Synopsis
Switch-OBSVirtualCam : ToggleVirtualCam

---
### Description

Toggles the state of the virtualcam output.


Switch-OBSVirtualCam calls the OBS WebSocket with a request of type ToggleVirtualCam.
