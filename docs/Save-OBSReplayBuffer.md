Save-OBSReplayBuffer
--------------------
### Synopsis
Save-OBSReplayBuffer : SaveReplayBuffer

---
### Description

Saves the contents of the replay buffer output.


Save-OBSReplayBuffer calls the OBS WebSocket with a request of type SaveReplayBuffer.
