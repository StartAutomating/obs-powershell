Get-OBSSceneCollection
----------------------
### Synopsis
Get-OBSSceneCollection : GetSceneCollectionList

---
### Description

Gets an array of all scene collections


Get-OBSSceneCollection calls the OBS WebSocket with a request of type GetSceneCollectionList.
