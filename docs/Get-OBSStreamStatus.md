Get-OBSStreamStatus
-------------------
### Synopsis
Get-OBSStreamStatus : GetStreamStatus

---
### Description

Gets the status of the stream output.


Get-OBSStreamStatus calls the OBS WebSocket with a request of type GetStreamStatus.
