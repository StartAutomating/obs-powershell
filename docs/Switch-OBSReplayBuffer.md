Switch-OBSReplayBuffer
----------------------
### Synopsis
Switch-OBSReplayBuffer : ToggleReplayBuffer

---
### Description

Toggles the state of the replay buffer output.


Switch-OBSReplayBuffer calls the OBS WebSocket with a request of type ToggleReplayBuffer.
