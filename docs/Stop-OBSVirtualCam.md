Stop-OBSVirtualCam
------------------
### Synopsis
Stop-OBSVirtualCam : StopVirtualCam

---
### Description

Stops the virtualcam output.


Stop-OBSVirtualCam calls the OBS WebSocket with a request of type StopVirtualCam.
